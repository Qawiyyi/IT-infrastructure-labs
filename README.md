# Active Directory Home Lab

## Overview
Built a Windows Server 2022 Active Directory lab using VirtualBox and Windows 10 client machines.

## Technologies Used
- Windows Server 2022
- Active Directory Domain Services
- DNS
- VirtualBox
- Windows 10
- PowerShell
- Networking

## Key Configurations
- Configured static IP addressing
- Installed AD DS and DNS
- Created Organizational Units and users
- Joined Windows 10 client to domain
- Tested domain authentication
