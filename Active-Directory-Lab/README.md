# Active Directory Home Lab

## Overview

This project shows how I set up a Windows Server 2022 Domain Controller and created a basic enterprise environment using Active Directory.

## Objectives

* Install Windows Server
* Configure static IP and DNS
* Install Active Directory
* Promote server to Domain Controller
* Create users and manage access

## Tools Used

* Windows Server 2022
* VirtualBox
